# 🍽 Smart Restaurant Ordering System

A QR-Based Restaurant Ordering System developed using Spring Boot, Thymeleaf and MySQL.

## 📌 Project Overview

This project allows restaurant customers to scan a QR code placed on their table, browse the menu, add food items to the cart, place orders, and send them directly to the kitchen. The kitchen staff can update the order status, and the system also provides a sales report.

---

## 🚀 Features

- QR-Based Table Ordering
- Customer Details
- Digital Menu
- Food Search
- Shopping Cart
- Checkout
- Kitchen Dashboard
- Order Status Management
- Sales Report
- Responsive User Interface

## ▶️ How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Configure MySQL.
4. Run `RestaurantOrderingSystemApplication`.
5. Open:

```
http://localhost:8081/
```
